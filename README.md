# projeto-pid-controle-2
This repository contains code for a digital control system developed and implemented on hardware at Universidade Federal de São Carlos


## Compilar no linux:

Você deve ter instalado o `cmake` e o `make`

```bash
cmake CMakeLists.txt
make
./wall-e
```
## Compilar no windows:

Você deve ter instalado o `cmake` (caso não tenha, instale através desse [link](https://cmake.org/download/))

Vá para a pasta correta e rode os comandos:

```bash
cmake CMakeLists.txt
cmake --build . --config Release
./bin/Release/wall-e.exe
```
